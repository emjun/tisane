tisane.variable.AbstractVariable.associates_with¶

method

AbstractVariable.associates_with(variable)¶

Adds a correlation relationship to a data variable.

Parameters

variable ("AbstractVariable") – The variable that this variable is associated with/correlated to.

See also

causes

add a causal relationship

Examples

>>> import tisane as ts
>>> adult = ts.Unit(name="adult", cardinality=386)
>>> pounds_lost = adult.numeric(name="pounds_lost")
>>> age = adult.numeric(name="age")
>>> age.associates_with(pounds_lost)
